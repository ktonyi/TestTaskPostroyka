Приложение написоно на PHP 8.1 с использованием HTML.
На бекенде не прописана валидация.
Запуск производиться через form.html. 
        ( Все данные для ввода в форме являются обязательным,
        поля email и Имя - максимум 25 символов,
        телефоны строго 12 символов и начинается с 375.
        )
После ввода нажать на кнопу Сгенерировать
Результат : 2 подписи 1 личности в разном цвете.
